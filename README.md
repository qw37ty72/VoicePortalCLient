# Voice Portal

Игровой голосовой/видео сервис в стиле TeamSpeak и Discord: серверы с каналами, личные звонки, чат, друзья, стрим экрана, передача файлов до 150 ГБ, регистрация через Telegram.

## Стек

- **Сервер:** Node.js, Express, Socket.io, SQLite, Telegram Bot API
- **Клиент:** Electron, React, Vite, Framer Motion
- **Стиль:** тёмная неоновая тема, анимации (можно отключить в настройках)

## Запуск

### 1. Установка

```bash
cd d:\VoicePortal
npm install
cd server && npm install && cd ..
cd client && npm install && cd ..
```

### 2. Сервер

```bash
npm run server
```

Сервер: `http://localhost:3001`

### 3. Telegram-бот (регистрация)

В отдельном терминале:

```bash
npm run bot
```

В Telegram найдите бота по токену (или добавьте бота по username, если настроили в BotFather). Команда **/register** — получите User ID для входа в клиент.

### 4. Клиент

```bash
npm run client
```

Откроется Electron-окно. На экране входа введите **User ID** из бота и URL сервера (по умолчанию `http://localhost:3001`).

## Возможности

- **Регистрация:** только через Telegram-бота (команда /register).
- **Серверы и каналы:** как в TeamSpeak — создание серверов, голосовых каналов, чат в канале.
- **Друзья:** добавление по ID, личные сообщения (DM), **личные звонки** (как в Discord).
- **Голос:** шумоподавление (getUserMedia: noiseSuppression, echoCancellation, autoGainControl), мут микрофона и мут наушников (при муте наушников мутится и микрофон).
- **Веб-камера:** кнопка в нижней панели.
- **Стрим экрана/приложения:** окно выбора источника (экран или окно), качество до 4K, частота кадров до 120 FPS (в Electron — выбор конкретного окна/экрана).
- **Передача файлов:** до **150 ГБ** на файл, чанками по 50 МБ.
- **Тема:** тёмная неоновая, анимации можно отключить в **Настройки** (иконка шестерёнки в сайдбаре).

## Токен бота

Токен бота задаётся в коде (`server/src/bot/index.js`) или через переменную окружения `TELEGRAM_BOT_TOKEN`.  
**Важно:** после публикации токена в открытом доступе его лучше сменить в BotFather и прописать новый в конфиге/окружении.

## Структура

- `server/` — API, WebSocket (чат, сигналинг WebRTC), загрузка файлов, бот.
- `client/` — Electron + React: логин, серверы/каналы, друзья, чат, звонки, стрим, файлы, настройки.
